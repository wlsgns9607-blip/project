WHY PLANNET KITCHEN

주방은 여전히,
암산과 고함으로 돌아간다

ORDER 01

동적 변수를 놓치는 아날로그 타임테이블

코스 중 한 테이블이 2~3분만 밀려도 뒤이어 나갈 재료는 오버쿡되거나 식는다. 그런데도 전체 타임라인은 실시간으로 재연산되지 않아, 그 부담은 고스란히 셰프의 뇌로 향한다.

ORDER 02

감(感)에 기대는 R&D, 엑셀로 버티는 원가 계산

맛의 조합은 순전히 경험에 의존하고, 손질 시 버려지는 손실률과 1인분당 소수점 단위 원가는 매번 엑셀 노가다로 되풀이된다. 창작에 쓸 시간이 행정에 소진된다.

ORDER 03

고함으로 메우는 파트 간 소통

메인 패스와 Cold·Hot·Pastry 파트의 진행 상황이 동기화되지 않는다. 피크 타임엔 정보가 꼬이고, 주방은 목소리 크기로 문제를 해결한다.

AS-IS → TO-BE

AI 대시보드로

AS-IS
  • 매직으로 빼곡히 채운 화이트보드와 수십 개의 자석 스톱워치
  • 선배에게 물려받은 엑셀 수식, 인스타그램 이미지 벤치마킹
  • 인이어와 대면 고함으로 전달되는 홀-주방 커뮤니케이션
→
TO-BE
  • 실시간 가변형 역산 알고리즘 — 지연 발생 시 버튼 하나로 전체 코스 타이머 자동 재조정
  • AI 퀴진 엔진 — 분자 페어링 추천과 저울 연동 수율·원가 실시간 대시보드
  • Section Sync — 파트별 카드 큐레이션과 DONE 터치 한 번의 초록불 동기화

CORE SYSTEM

3대 주요 기능

FEATURE 01

AI 가변형 역산 타임테이블

코스 전체 흐름을 역산해 미리 배치하고, 라이브 지연이 발생하면 전체 워크플로우를 동적으로 재연산한다. [AI 타임라인 재연산] 버튼 하나가 뒤이어 나갈 모든 조리 타이머를 자동으로 연동·조정해 최적의 서빙 온도를 지킨다.

FEATURE 02

AI 퀴진 랩 — 분자 페어링 & 스마트 수율

식품 공학 데이터 기반 Flavor Compounds로 맛 조합을 추천하고, 스마트 저울과 연동해 원물 대비 가용 무게 수율을 자동 계산한다. 목표 마진율을 입력하면 원가를 역산하는 시뮬레이터까지 하나의 흐름으로 이어진다.

FEATURE 03

멀티 엔딩 워크플로우 — 파트 싱크

주방 세션(Cold·Hot·Pastry)별 태블릿에는 전체 흐름 대신 '지금 내가 할 일'만 카드로 큐레이션된다. 초 단위 카운트다운 푸시와 인터랙티브 DONE 버튼이 메인 대시보드에 실시간 초록불로 동기화된다.

SECTION SYNC

DONE — 고함 없는 초록불

Hot 파트원이 태블릿 카운트다운을 보고 로스팅을 마친 뒤 [DONE]을 누른다. 메인 셰프 대시보드에 초록불이 실시간으로 동기화되며, 고함 없는 주방이 완성된다.
